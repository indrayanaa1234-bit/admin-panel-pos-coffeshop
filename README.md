# CoffeeShop POS Dashboard

Jalankan project menggunakan Live Server agar komponen navbar AJAX dapat dimuat dengan benar.

Halaman tersedia:
- POS: `index.html`
- Riwayat: `pages/history.html`
- Dashboard: `pages/dashboard.html`

## Halaman Laporan

Halaman laporan berada di `pages/report.html` dan membaca data transaksi dari LocalStorage key `transactions`. Fitur utama: filter Hari Ini/7 Hari/30 Hari/Semua, cetak laporan, export CSV, ringkasan statistik, grafik produk, grafik kategori, grafik metode pembayaran, dan detail transaksi terakhir.
